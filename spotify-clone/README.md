# spotify-clone

User-facing Spotify-style UI (playback, likes, playlists).

## Run (dev)

```bash
npm install
npm run dev
```

## Environment
By default the app uses `http://localhost:4000` for the API.

Optional: create `.env` in this folder:

```dotenv
VITE_API_URL="http://localhost:4000"
```
