# spotify-admin

Admin dashboard for managing songs/albums and admin accounts.

## Run (dev)

```bash
npm install
npm run dev
```

## Notes
- Expects the backend API to be running (default: `http://localhost:4000`).
- Stores admin auth token in `localStorage` (key: `admin_token`).
